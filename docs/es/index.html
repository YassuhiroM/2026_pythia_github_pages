Pythia

Ciencia de Datos para Cafeterías

Proyecto final del Máster en Ciencia de Datos en UCM/NTIC.

Proyecto real Utiliza datos reales de ventas y reseñas de clientes de una cafetería de Madrid para construir una aplicación de cafetería práctica e inteligente.

Integra distintos campos de conocimiento Combina de forma eficaz contenidos de ETL, EDA, Machine Learning, Deep Learning, Series Temporales, NLP y visualización avanzada en una única solución coherente.

Construido como producto real Presentado como un MVP funcional en Streamlit, mostrando el proyecto como un producto utilizable y no solo como una entrega académica.

Reconocimientos Obtuvo la calificación máxima y el 2.º puesto en un concurso de pitch.

Detalles del Proyecto

Pythia está diseñado para leerse de una sola vez: primero el reto de negocio, después el flujo analítico, luego las decisiones de modelado y, por último, el impacto medible.

Problema de Negocio

Las cafeterías de especialidad operan con márgenes ajustados, productos perecederos y una demanda muy variable. Este proyecto aborda ese reto combinando predicción de ventas y análisis de reseñas de clientes para apoyar mejores decisiones en compras, personal y prioridades de mejora del negocio.

Datos y Metodología

El proyecto combina datos estructurados y no estructurados, incluidos registros diarios de ventas, datos meteorológicos, calendarios de festivos, flujos de visitantes de CaixaForum y reseñas de Google. Siguiendo un enfoque CRISP-DM, integra predicción de series temporales y minería de texto en un único sistema de apoyo a la decisión para la cafetería.

Modelado

Para la predicción, se seleccionó Prophet tras compararlo con Regresión Lineal y XGBoost, al ofrecer el mejor equilibrio entre precisión, robustez e interpretabilidad. Para la minería de texto, el pipeline combina YAKE, NLTK y Sentence-BERT para extraer, normalizar y agrupar automáticamente los temas de feedback de clientes a partir de las reseñas.

Resultados e Impacto

Los modelos de predicción lograron resultados sólidos, con una predicción de ingresos de alrededor de 11,74% de MAPE en validación y 14,25% en monitorización en producción. El análisis de reseñas identificó fortalezas clave como la comida, el café y el servicio, y señaló el precio como la principal área de mejora. Ambos componentes se desplegaron en una aplicación Streamlit, convirtiendo el proyecto en una herramienta práctica para la toma de decisiones reales de negocio.
